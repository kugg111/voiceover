A free, open-source Discord alternative

Voiceover is a free voice chat app for Windows built as an open-source Discord alternative — the full source is public on GitHub, so you can read exactly what it does, self-host your own server, or contribute. No paid tiers, no ads, no account data sold: just servers, channels, direct messages, and voice chat with real end-to-end encryption. Made in Hungary by a solo developer.

FAQ

Is it actually private?

Channel messages and direct messages are end-to-end encrypted client-side (ECDH + AES-256-GCM) before they ever leave your device - the server only ever stores and relays ciphertext it has no key to read.

Can I self-host it?

Yes - the full source (server and client) is public on GitHub. It's an ASP.NET Core server plus a Postgres database, with voice/screen sharing routed through a self-hosted LiveKit deployment.

Is it free?

Yes, completely - no paid tiers, no ads, nothing held back behind a subscription.

Voiceover is an unsigned Windows app — you may see a SmartScreen warning on first launch; choose "More info" → "Run anyway".
