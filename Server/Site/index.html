🎧 Voiceover GitHub

A Discord-style app,
built from scratch.

Servers, text channels, direct messages, friends, and mesh voice chat — a lightweight desktop client for you and your friends.

⬇ Download for Windows Windows 10/11 · No installer needed, just unzip and run
💬

Servers & channels

Create servers, organize text and voice channels, invite friends with shareable codes.

🎙️

Voice chat

Live voice channels with a speaking indicator, powered by real peer-to-peer WebRTC audio.

✉️

Direct messages

1:1 conversations with a real-time inbox, separate from your servers.

🧑‍🤝‍🧑

Friends

Send and accept friend requests, see them update live.

Voiceover is an unsigned Windows app — you may see a SmartScreen warning on first launch; choose "More info" → "Run anyway".
